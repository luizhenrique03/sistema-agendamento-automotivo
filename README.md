# 🚗 Sistema de Agendamento Automotivo

Sistema web desenvolvido para auxiliar oficinas mecânicas no gerenciamento de agendamentos de serviços de forma simples, organizada e eficiente.

---

## 📌 Sobre o Projeto

O objetivo do sistema é facilitar o processo de marcação de serviços automotivos, ajudando tanto clientes quanto profissionais a manterem seus compromissos organizados.

A proposta foi criar uma solução funcional e prática, priorizando a lógica de negócio e a integração entre front-end e back-end.

---

## 🛠️ Tecnologias Utilizadas

- Python  
- JavaScript  
- HTML  
- CSS  

---

## ⚙️ Funcionalidades

- Cadastro de clientes  
- Agendamento de serviços  
- Organização de horários  
- Tratamento e manipulação de dados  
- Interface simples e funcional  

---

## 🧠 Conceitos Aplicados

- Integração entre front-end e back-end  
- Estruturação de lógica de negócio  
- Manipulação de dados com Python  
- Organização de projeto  
- Boas práticas de desenvolvimento  

---

## 🚀 Objetivo do Projeto

Mais do que apenas desenvolver um sistema, este projeto teve como foco reforçar conhecimentos em arquitetura de aplicações web, organização de código e construção de soluções práticas para problemas reais.
